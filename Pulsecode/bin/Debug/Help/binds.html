Keybinds

General

F1 - Open help menu
Ctrl + S - Save the chart
Up arrow key - Increase playback rate
Down arrow key - Reduce playback rate
Space - Pause/unpause the song

Scrolling

Left arrow key - Scroll backwards through the chart
Right arrow key - Scroll forwards through the chart
Shift + left/right arrow key - Scroll backwards/forwards at a rate of 4x
Ctrl + left/right arrow key - Scroll backwards/forwards at a rate of 0.5x
B + left/right arrow key - Scroll forwards or backwards through bookmarks

Note placement

Numbers 1 to 7 - Place a note on lane 1 to 7
H - Toggle whether the selected notes are hold notes
Shift while dragging a note - Snap it to nearest (snap / 2)
Ctrl + up/down arrow Key - Move the selected notes forward or backward by (snap / 2)
Delete - Delete any selected notes
